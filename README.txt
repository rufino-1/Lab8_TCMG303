C++
add.cpp
This program takes two integers from the user, adds the integers, and outputs the result
To run the code:
Check if g++ is installed. If it is installed, then you can run the following lines.
g++ add.cpp
./a.out
***
Example:
Enter the first integer: 5
Enter the second integer: 10
The result of adding 5 and 10 is: 15 
***
For information about installing GCC go to https://linuxize.com/post/how-to-install-gcc-compiler-on-ubuntu-18-04/

Python
goingToZero.py
This is a Python code. The program will ask for an integer from the user. 
If the integer is positive, it will output numbers decreasing by 1 until 0 is reached. 
If the input is negative, it will output numbers in increments of 1
To run the code:
Check if Python is installed (python --version). If Python is installed then you can run the code by using the next line
 python3 goingToZero.py
If we enter the integer 5. We will get the following output.
5 -> 4 -> 3 -> 2 -> 1 -> 0
For information about installing python go to https://docs.python-guide.org/starting/install3/linux/
